Grab All Files テストサイト

Google Chrome™ / Microsoft Edge™ 拡張機能「ファイル一括ダウンロード（Grab All Files）」の全検出パターン確認用

lib/file-utils.js の knownExtRe に列挙された 225 拡張子と、 popup.js / page-interceptor.js に実装された 40 以上の検出手法を網羅したテストフィクスチャです。 各ページを開いて拡張機能のポップアップから Scan を実行してください。

① HTML 要素 / 属性検出

01. 基本リンク <a href> + download / type 属性 02. 画像（lazy / srcset / picture / CSS背景） 遅延読み込み・レスポンシブ画像・CSS background 03. 埋め込み iframe / embed / object 04. data-* 属性（25種） data-href, data-pdf-url, data-zoom-image など全網羅

② メタ・スクリプト・フォーム

05. メタタグ・インラインスクリプト OGP / Twitter / citation_pdf_url / pdfUrl 等 17 キー / JSON-LD 06. POSTフォーム / hidden input 「ダウンロード」ボタン (JP/EN/CN) / ポータル風 hidden / 除外条件 07. blob: / fetch / XHR ネットワーク監視（page-interceptor.js） / 遅延 DOM 注入

③ URL推定パターン

08. 拡張子なしURL（クラウド・LMS・CMS） S3 / GCS / Moodle / Canvas / Liferay / 文書ホスティング等 12. PDF.js ビューア・PDF閲覧URL /_pdf/, /viewer/pdf, response-content-type, e-manual

④ 高度な検出

11. プラットフォーム特化（Gmail / Yahoo / Outlook / Salesforce） download_url 属性 / disp=safe→attd / aria-label / 069 ContentDocument 14. マジックバイト判定 拡張子と中身がミスマッチ → 先頭バイトから種類判定 15. CSS スキャン backgroundImage / image-set / border-image / @font-face / cursor 09. 誤検出トラップ SourceForge / naturalearthdata 等、除外されるべきパターン

⑤ 全拡張子・クロール深度

10. 全 225 拡張子の網羅リンク knownExtRe に列挙されたすべての対応拡張子 16. Deep Scan チェーン (level-1 → 3) クロール深度ごとに到達できるファイル数が増加
Grab All Files テストフィクスチャ。検出ロジックは lib/file-utils.js / popup.js / page-interceptor.js、 機能仕様は docs/sales-feature-document-ja.md を参照。
